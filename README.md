# Flutter Grocery App

A Flutter-based grocery shopping mobile application developed as part of a learning project.

## Features

- User login and signup flow
- Home screen with grocery categories
- Product listing and product details
- Add to cart functionality
- Cart and checkout flow
- Order success screen
- Past orders tracking
- Favorite products
- Notifications
- Profile screen with address support
- Dark mode and light mode switching
- Clean Git setup with `.gitignore`

## Technologies Used

- Flutter
- Dart
- SharedPreferences
- Git and GitHub
